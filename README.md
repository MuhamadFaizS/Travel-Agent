# AI TravelBot (Travel Booking Assistant) 🛫

This project is an AI-powered Travel Assistant built using **LangChain**, **LangGraph**, and **Google Gemini 2.5 Flash Lite**. The bot acts as an interactive travel agent that helps users explore a travel catalog, build an itinerary, and finalize their bookings.

## 🚀 Use Case & Creative Parameters
- **Use Case:** Travel Assistant (Virtual Agent for booking flights, hotels, and tours).
- **Style:** Friendly, professional, and proactive.
- **Features:** 
  - Memory capabilities via LangGraph `InMemorySaver` to remember user itineraries.
  - Interactive CLI UI.
  - Mock database integration via LangChain Tools.
